# Stream Owl

A movie recommendations and finder application built with Deno Fresh.

## Quick Start

### Development

1. Clone the repository
2. Copy `.env.example` to `.env` and configure environment variables
3. Run database migrations: `deno task migrate`
4. Start development server: `deno task start`

### Deployment

See [Deployment Guide](docs/DEPLOYMENT.md) for instructions on deploying to Deno
Deploy.
